<!-- GETTING STARTED -->
## Getting Started



### Prerequisites


* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/amithjayapraban/p2p.git
   ```
1. Install NPM packages
   ```sh
   npm install
   ```
